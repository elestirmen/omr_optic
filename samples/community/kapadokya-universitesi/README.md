# Kapadokya Üniversitesi Optik Formu (Temel Şablon)

Bu klasördeki `template.json`, paylaştığınız Kapadokya Üniversitesi optik form düzenini temel alır:

- `AdSoyad` (TR alfabe), `TCKimlikNo` (11 hane), `OgrenciNo` (9 hane), `Telefon` (3+7 hane)
- `kitapcik` (A/B/C/D)
- `q1..120` (A/B/C/D/E)

Kendi taramalarınız için bu şablonu, görsellerin bulunduğu klasör ağacına `template.json` olarak kopyalayın (ör. `inputs/template.json`) ve gerekiyorsa `--setLayout` ile `origin`, `bubblesGap`, `labelsGap` değerlerini ince ayar yapın.
